# chatter-box
A simple app used to send messages between users


# How to run locally?
1) Naviagate to the required folder
2) git clone https://github.com/dinudines/chatter-box.git
3) cd chatter-box
4) npm install
5) node app.js

# Features 
1) send one to one messages
